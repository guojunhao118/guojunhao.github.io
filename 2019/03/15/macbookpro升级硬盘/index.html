macbookpro升级硬盘

发表于 2019-03-15 |

之前一直使用的是15款低配的 macbookpro（m839） 硬盘容量问题自是苦不堪言，常常是捉襟见肘，最近看到同事更换固态硬盘，便也有了更换大容量的这么一个想法，研究一番，也准备动手实践一下，此文权当是记录，也算做是踩坑指南吧

前期准备

  • 测试硬盘速度（前后对比）

Disk Speed Test

  • 查看链接宽度（前后对比）

正常情况下链接宽度是X4

  • 转接卡

还是根据机油们的推荐，某宝 迪奥科技购买即可，或者自行搜索M.2 NGFF 转 MacBook

保险起见，我购买了两个转接卡，店主还帮忙配了些绝缘胶带

  • 螺丝刀

拆开 MacBook 底盖所需要的工具为：1.2 5角 螺丝刀。（关键字直接搜索可找到对应商品，下同）

卸下原装 SSD 所需的工具为：T5 梅花 螺丝刀。进行这一步操作之前要记得断开电池连接主板的排线。

  • 固态硬盘
  1. 原装，贵
  2. M.2标准接口SSD（NVME）通用型强
  3. 在 macOS 10.13 正式发布之后，支持原生 NVME 协议，因此更换上相应的固态硬盘成为可能

综上所述

关于固态硬盘，其实并没有太多的选择，就直接买了大多数人推荐的 intel 760p(NVME)

也有网友推荐使用三星SM951a，这里略去不提，可以自行百度

按照官方教程走就可以了（这里注意要将系统升级到最新的 macos ）

注意的是先左边再右边的顶开后盖

  • 断开电源

  • 安装转接卡

安装过程

  • 安装系统
  1. 重启计算机，按住 option 键不放，便可以看到相应的安装盘选项，选择相应的设备即可。
  2. 选择磁盘工具-继续
  3. 选择左上角的“内置”SSD，点击“磁盘工具>抹掉”，在“名称”中自定义名称（此名称在系统启动后可随时更改），“格式”选择“Mac OS 扩展（日志式）”，方案选“GUID分区图”，点击“抹掉”，然后点“完成”
  4. 安装前要抹掉先前的磁盘，这里格式选择 AFPS，方案为 GUID 分区图。
  5. 接着等待安装完成，重新激活开机。
  6. 重置SMC 和 NVRAM

用Mac OS的启动U盘全新安装完系统之后，记得要重置一下SMC 和 NVRAM，方法见苹果官方网站：

重置SMC的方法：如何重置 Mac 上的系统管理控制器 (SMC)

重置NVRAM的方法： 如何重置 Mac 上的 NVRAM
