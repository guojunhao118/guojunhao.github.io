DOM操作

方法 说明
var haha = document.querySelector('#id') 查找元素(单个)
var hehe = document.querySelectorAll('#id') 查找元素（所有）
var hahaValue = haha.getAttribute(‘value’) 获得特定属性值
haha.setAttribute(‘value’, ‘新用户名’) 设置特定属性值
haha.hasAttributes() 查看元素是否有属性
haha.hasAttribute('value') 查看元素是否有特定属性
haha.removeAttribute('type') 删除 某个属性
var button = document.createElement('button') 创建元素
button.innerHTML = ‘注册用户’ 用 innerHTML 设置属性
form.appendChild(button) 添加元素(from表示选出来的元素,button是刚才创建的按钮)
haha.remove( ) 自毁
haha.removeChild( ) 父节点删除子元素
haha.insertAdjacentHTML( 'beforeend', html ) 插入 HTML
insertAdjacentHTML 标签 描述
beforebegin 开始标记 前
afterbegin 开始标记 后
beforeend 结束标记 前
afterend 结束标记 后
