SQL 语句

语句 说明 例子
SELECT DISTINCT 列出不同值 SELECT DISTINCT 列名称 FROM 表名称
SQL WHERE 有条件地从表中选取数据 SELECT 列名称 FROM 表名称 WHERE 列 运算符 值
SQL AND & OR 对记录进行过滤 SELECT * FROM Persons WHERE FirstName='Thomas' AND LastName='Carter'
SELECT * FROM Persons WHERE firstname='Thomas' OR lastname='Carter'
SELECT * FROM Persons WHERE (FirstName='Thomas' OR FirstName='William') AND LastName='Carter'
SQL ORDER BY 对结果集进行排序
SQL INSERT INTO 用于向表格中插入新的行

SQL查询语句

语法

基本查询

SELECT 列名称 FROM 表名称 / SELECT * FROM 表名称

模糊查询

select * from club_member where name like "%20170306%";

查总数

select count(*) from club_member where id > 2;

SQL查询语句的顺序：SELECT、FROM、WHERE、GROUP BY、HAVING、ORDER BY。SELECT、FROM是必须的，HAVING子句只能与GROUP BY搭配使用。

数据库是应用最广泛的计算机软件

概述

数据库现在主要分 关系型数据库（传统） 和 NoSQL（新式比如 mongodb）和一些其他数据库（比如 fb 的图数据库）
～　常用的关系型数据库有 mysql postgresql sqlite 等

几种关系型数据库的用法和 sql 语法都极度相似
开发中一般会用 sqlite 数据库
部署到服务器上的时候才会使用 mysql 等数据库
当然 mysql 已经不适合这个版本了
对于 node 来说, mongo 是非常方便的
MEAN 技术栈 = mongo + express + angular + node

mongodb

安装

sqlite 直接用 yarn 安装模块
并且需要安装一个叫 sqlitebrowser 的管理软件

mongodb 需要安装，链接如下
软件本身
http://www.runoob.com/mongodb/mongodb-window-install.html
管理软件
https://robomongo.org/download

mac 下安装 mongo

  1. brew install mongodb
  2. brew cask install launchrocket
  3. 在[设置]底部那一栏里找到 launchrocket, 点击进去之后, 就可以看到 mongodb 了
    点击 start 就可以了

window 下 yarn install better-sqlite3 失败的解决方式

  1. 用管理员身份打开命令行, 切换到项目目录
  2. yarn global add windows-build-tools
  3. 安装 python2.7 并且设置环境变量
  4. yarn install

如果还是失败的话, 从 http://landinghub.visualstudio.com/visual-cpp-build-tools 下载 Visual C++ Build Tools 2015 并且安装
然后执行 yarn install
